JOEL - painless task management

Joel was developed to help you keeping track of your Todos and the time you spent on it in a painless and unobstrusive way. It is inspired by Joel Spolskys 4 column speadsheet.

Software and Libraries used to build Joel:
Javascript, Flash AS3, Trimline JST, Google Gears, JSON, Mootools, Firebug, PHP, ADOdb lite,

   
Tasklist shortcuts

ctrl+shift+c copy selected task
ctrl+shift+x cut selected task
ctrl+shift+v paste copied/cut task at current position
ctrl+backspace delete selected task(s)
shift+click select tasks between clicked and last selected
ctrl+click select clicked task additionally to last task(s)
enter to create and select a new task
up select previous task
down select next task
ctrl+shift+h switch tasks to block heading and vice versa
ctrl+shift+d press multiple times to add date, time or date and time
ctrl+shift+i increase task priority by 1
ctrl+shift+up add 5 minutes to elapsed time of current task
ctrl+shift+down take 5 minutes from elapsed time of current task


Note: on Apple Computers use the "apple" key instead of ctrl
